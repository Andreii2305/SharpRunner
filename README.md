# SharpRunner

_A Gamified Learning Platform for Mastering C# Fundamentals_

SharpRunner is an interactive, beginner-friendly web app designed to help students learn C# Fundamentals through short lessons, challenges by playing a 2D sidescroller platform. Instead of traditional lectures, SharpRunner breaks down the core concept into small, playable activities to make learning easier and more enjoyable.

## Project Goal

The main goal of SharpRunner is to make learning programming less intimidating. It focuses on the fundamentals of C#.

## Features

#### Beginner-Friendly Lessons

Covers essential topis such as: - Variables and Data Types - Operators - Conditional Statements - Loops

Each Lesson includes an explanation.

#### Gamified Learning

    - Level-Based Lesson
    - Challenges and task to reinforce understanding
    - Embeded code editor / compiler, so user can control the character to achieve task and challenges

#### Clean and Simple UI

Design to help learners focus, especially younger students or programming beginners.

## Interfaces

#### Student Interface

Student can login, Join Classroom with classroom code provided by instructor.
Play the game in correspond with each lesson and level.
Learn.
Track their own progress.
View Leaderboard in their class and compete with other classmate.

#### Instructor Interface

Instructor can also login, and create a classroom for their handled section.
Classroom code will be automatically generated upon creation.
Accept or allow student to join.
Track all students from different sections handled.
Track specific classroom performances.
Track student performances.
Then adapt based on analytics.

#### Admin Interface

Admin can create an account for Instructor, Student, and for other Admin.
Can manage users (remove, add, hold).
Can accept users when they request to create an account.
Create a classroom.
Monitor the system logs.

## Tech Stack

    - React.js      : Front-end Framework
    - Node.js       : backend logic and interactivity
    - CSS           : Styling
